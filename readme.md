## Includes:

* create-react-app
* storybook with addons (actions, options, chapters)
* koa
* antd

## Clone

```bash
git clone https://github.com/pomber/react-dx-boilerplate.git my-project
cd my-project
rm -rf .git
yarn install
yarn dev
```

## Usage

* Move between stories with `ctrl + shift + right` or `ctrl + shift + left`
